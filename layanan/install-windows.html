Layanan Utama

Install Ulang Windows, Siap Pakai.

Format bersih atau upgrade ke versi terbaru. Semua driver terinstall, Windows aktif, Office siap pakai, dan laptop langsung fungsional.

Apa yang kamu dapatkan

  • Install Windows 7, 8, 8.1, 10, atau 11 sesuai pilihan
  • Driver lengkap: chipset, LAN, WiFi, audio, GPU
  • Microsoft Office (Word, Excel, PowerPoint)
  • Browser: Google Chrome / Mozilla Firefox
  • PDF Reader, Multimedia Player
  • .NET Framework, DirectX, Visual C++ Redistributables
  • Test fungsionalitas lengkap sebelum diserahin

Estimasi Waktu

Reguler: 1–2 hari kerja. Express: selesai dalam 24 jam (+Rp 25.000).

Harga

Rp 50.000 flat untuk install ulang Windows standar. Belum termasuk biaya antar-jemput jika menggunakan layanan jemput.

Garansi

Garansi 7 hari. Kalau ada masalah teknis setelah install, bisa claim garansi tanpa biaya tambahan.

Lokasi & Pengiriman

Teknisi berbasis di Daerah Istimewa Yogyakarta. Kamu bisa:

  • Antar & ambil sendiri ke lokasi teknisi — gratis
  • Teknisi jemput ke lokasi kamu — Rp 2.000/km (satu arah)
  • Jemput & antar (PP) — Rp 2.000/km × 2
  • Gojek/kurir — biaya sesuai tarif aplikasi
